# 💊 Pharmacy Management System

## 📖 Project Overview

The **Pharmacy Management System** is a desktop application developed using **C# Windows Forms** and **SQL Server**. The system is designed to simplify daily pharmacy operations by organizing medicine records, stock information, supplier details, purchases, and sales. It provides separate modules for **Admin**, **Pharmacist**, and **Cashier**, allowing each user to perform their assigned responsibilities efficiently.

---

## ✨ Features

- 👨‍💼 Admin Module
- 💊 Pharmacist Module
- 💰 Cashier Module
- Category Management
- Medicine Management
- Supplier Management
- Batch Stock Management
- Purchase Management
- Purchase Item Management
- Sales Management
- Sales Item Management
- Inventory Management
- CRUD (Create, Read, Update, Delete) Operations
- SQL Server Database Integration
- User-Friendly Windows Forms Interface

## 📂 Modules

### 👨‍💼 Admin
